Emacs Commands
==============

1. C-x k
    * kills a buffer
2. C-x C-s
    * saves a buffer
3. C-v
    * goes down a page
4. C-l
    * redraws screen and puts line you are on in middle
5. C-k
    * kills the rest of the line you are on
6. C-x o
    * opens new file
7. C-h
    * opens help
8. C-y
    * yanks from kill ring
9. C-s
    * searches
10. C-x C-c
    * quits emacs
